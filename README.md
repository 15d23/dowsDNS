# dowsDNS

快速翻跃中国防火墙

# 使用方法

修改文件config.json

`"Local_dns_server" : "10.0.2.15",`

`10.0.2.15` 把此IP换成本地IP,或者公网IP

然后执行 `python dns.py`

修改本机的dns地址为本机ip地址即可
